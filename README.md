# ABC Live API

ABC Live API is the API made for the BFSI news compiling website [ABC Live Web](https://abclive-bfsi.vercel.app). The API collects headlines with other information like description and hyperlink from ET BFSI and Business Line, and important values of rates from RBI.

Links for all API pages are follows:  
ET BFSI: https://abc-live-web-api.vercel.app/et_bfsi  
Business Line: https://abc-live-web-api.vercel.app/business_line  
RBI: https://abc-live-web-api.vercel.app/rbi
